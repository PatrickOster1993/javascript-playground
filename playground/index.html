JavaScript Playground

Code befindet sich innerhalb Ihres Browser-eigenen Entwicklertools unter der Rubrik "Console"

  1. Öffnen Sie die Datei "index.html":
    1. Gehen Sie im Datei-Explorer zum Projektordner, in dem sich "index.html" befindet.
    2. Klicken Sie mit der rechten Maustaste auf "index.html".
    3. Wählen Sie "Öffnen mit" und dann "Google Chrome" aus der Liste der Anwendungen.
  2. Klicken Sie mit der rechten Maustaste auf die Seite und wählen Sie "Untersuchen" aus dem Kontextmenü.
  3. Alternativ können Sie auch die Tastenkombination Strg + Umschalt + I (Windows/Linux) oder Befehl + Wahltaste + I (Mac) verwenden.
  4. Das Entwicklertool wird am unteren oder rechten Rand des Bildschirms angezeigt.
  5. Wechseln Sie zur Registerkarte "Console", um das Ergebnis innerhalb der Console zu sehen.
